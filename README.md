# 🏋️ GymSync-CLI

GymSync-CLI is a high-performance, interactive command-line automation suite built with Python and **Selenium WebDriver**[cite: 1]. Designed to bypass bulky graphical user interfaces, it provides a fast, ANSI-colored terminal dashboard to control, track, and sync custom fitness class schedules directly into live browser instances[cite: 1].

---

## 🔥 Features of My App

* **Zero-File Local Storage Persistence:** Saves user configurations directly inside Google Chrome's native `localStorage` layer[cite: 1]. No annoying JSON or configuration files will ever clutter your desktop or workspace directory[cite: 1].
* **Intelligent Account Exception Handling:** Actively monitors the web application's layout state[cite: 1]. If an account isn't found or an invalid state occurs, it intercepts the exception, prompts you to virtualize registration on-the-fly, or cleanly closes with a parting message (`Ok byeee........`)[cite: 1].
* **Live DOM Manipulation Engine:** Seamlessly updates the browser interface by injecting custom calendar blocks, custom actions, and localized status layouts via asynchronous JavaScript runtime loops[cite: 1].
* **Dedicated Security Teardown (Logout Module):** Features a total state-clear engine that completely purges saved data matrices, flushes local storage tracking blocks, and cleanly resets your authentication environment[cite: 1].
* **Locked Profile Guard:** Automatically provisions isolated persistent session folders (`chrome_profile`)[cite: 1]. It handles multi-instance hardware blocks gracefully, instructing you to safely restart if a session clash occurs[cite: 1].
* **Clean Traceback Shielding:** Intercepts system interrupts (double `Ctrl+C`) to terminate code execution natively, keeping your terminal window clean and free from ugly Python error stacks[cite: 1].
